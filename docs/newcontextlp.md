🛠️ Prompt de Estruturação: Landing Page KarCash
Contexto:
Aja como um Especialista em UX/UI e Conversão. O objetivo é criar uma Landing Page Mobile-First para o projeto KarCash, uma comunidade VIP de ofertas de carros com margem de lucro (20% a 50% abaixo da FIPE). A estrutura deve ser inspirada no fluxo de conversão do clubedocarroapp.com.br, utilizando a stack React + Tailwind CSS.

Requisitos de Seções (Ordem de Escaneabilidade):

Hero Section (O Gatilho):

Headline: Focada em lucro imediato e autoridade. Ex: "Acesse as ofertas que as concessionárias escondem e lucre até R$ 20 mil por carro."

Subheadline: Explicar que não precisa de CNPJ e o acesso é imediato.

CTA: Botão flutuante ou central "QUERO ACESSO VIP".

Visual: Mockup do celular mostrando o grupo VIP/App.

Trust Bar (Quebra de Objeção):

Ícones de "Pagamento Seguro", "Sem Fidelidade" e "Compra Direta com a Loja". Ênfase que o usuário não paga nada ao KarCash pelo veículo, apenas pela informação.

Vitrine de Desejo (Showcase):

Carrossel com 4-6 cards de carros reais (Ex: Corolla, Compass, Onix).

Dados no card: Preço FIPE vs. Preço KarCash + "Margem de Lucro Estimada".

How It Works (Educação):

Passo a passo visual (1 a 4): Assine > Entre no Grupo > Escolha o Carro > Negocie Direto com a Concessionária.

Prova Social & Autoridade:

Seção de depoimentos (prints de WhatsApp) e contadores de "Membros Ativos" e "Economia Gerada para a Comunidade".

Checkout / Oferta Irresistível:

Ancoragem de preço forte (Ex: de R$ 197 por R$ 59,90).

Formulário de captura integrado pedindo Nome, E-mail, WhatsApp e CPF/CNPJ (obrigatório para o banco de dados Supabase).

FAQ (O Fechamento):

Acordeão respondendo: "Posso cancelar?", "Preciso de CNPJ?", "Como recebo o acesso?".

Footer Estratégico:

Links de Termos de Uso e Política de Privacidade. Disclaimer sobre a responsabilidade das negociações.

Diretrizes de Design:

Tom: Profissional, técnico e lucrativo.

Cores: Estilo "Dark Mode" ou azul profundo para transmitir confiança e exclusividade.

Performance: Código limpo para deploy na Vercel com carregamento instantâneo.
